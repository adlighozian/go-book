# Tipe Data Integer

### Integer

Tipe data integer adalah tipe data kumpulan number, tipe data integer direpresentasikan dengan kata kunci 'int' lalu diikuti dengan nilainnya.

Contoh code:

```go
var age int 

age := 23
	
fmt.Println("LastName")
```
